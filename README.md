# Detecting AI-generated Python Code

> 🏆 **Best Project among Bachelor & Master Teams** – Advanced Machine Learning, CUHK-SZ 
> *Slavinskaia Elizaveta, LI Yiran, LI Bingrong, WANG Xun, LIANG Ruoyu**

This project explores whether Python code was written by a human or generated by AI.

We built a multimodal detector that combines two views of the same code:

- **CodeBERT** — semantic understanding of code  
- **Graph Neural Network (GNN)** — structural analysis via AST  
- **Cross-modal attention** — fusion of both representations  

The final model achieved **93.31% accuracy** and **93.68% F1-score**, outperforming both CodeBERT-only and GNN-only baselines. More details in the [full report](report.pdf).

## Model Architecture

The model processes code in two parallel ways:

1. **Semantic branch**
   - Tokenized code → CodeBERT
   - Captures patterns in naming, style, and logic

2. **Structural branch**
   - Code → AST → Graph
   - Processed with Graph Attention Network (GAT)

3. **Fusion**
   - Cross-modal attention aligns both representations
   - Final classifier predicts AI vs Human
